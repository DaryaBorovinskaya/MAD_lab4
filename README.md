## Инструкция по запуску
- Если просто запустить и ничего не менять:
```
python app.py
```
- Если нужно переобучить модель K-Means:
```
python calculate_clusters.py
python app.py
```
- Если нужно переобучить модель PCA или были внесены изменения в препроцессинг:
```
python train_model.py
python train_trees.py
python app.py
```

## Файлы и их назначение
- app.py <br>
Главный файл, где запускается дашборд
- data_processing.py<br>
Загрузка, очистка и подготовка данных
- visualizations.py<br>
Функции для создания всех визуализаций
- modeling.py<br>
Содержит логику для обучения моделей и вычисления метрик (логистическая регрессия)
- callbacks.py<br>
Содержит все функции обратного вызова (callbacks) для Dash
